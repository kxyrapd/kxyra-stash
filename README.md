# kxyradev-stash

FiveM için geliştirilmiş, QB-Core uyumlu basit ve optimize edilmiş vector stash sistemidir.

---

##  Özellikler
- QB-Core uyumlu
- Vector tabanlı stash sistemi
- Optimize (düşük ms)
- Kolay kurulum
- Config üzerinden ayarlanabilir
- Client / Server ayrımı

---

##  Gereksinimler
- qb-core
- qb/ox inventory (veya hangi envanteri kullanıyorsan)
- oxmysql veya mysql-async

---

##  Kurulum
1. kxyradev-stash klasörünü resources dizinine atın
2. server.cfg içerisine ekleyin:
3. ensure kxyradev-stash
